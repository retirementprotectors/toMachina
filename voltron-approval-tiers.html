How It Works

When VOLTRON wants to execute a tool, the requiresApproval(toolName, userLevel) function checks these tiers before any action is taken. The check happens on every single tool invocation — no exceptions, no bypasses outside the rules below.

RED  tools always pause and ask for human approval — no matter who the user is. These cover outbound communications, client/account mutations, and bulk sends. PHI risk, brand risk, irreversibility.

YELLOW  tools require approval for team members (level 1+), but OWNER (JDM, level 0) bypasses automatically. These are admin-level operations JDM runs routinely — workspace management, WordPress, Canva, campaign scheduling, Drive mutations.

GREEN  tools execute immediately with no approval needed. Pure read-only lookups, searches, and status checks. VOLTRON chains these freely to gather context without interrupting the user.

Decision Logic — requiresApproval(toolName, userLevel)

Every tool call passes through this 5-step decision chain in order. First match wins. Source: voltron-os-rules.ts

1
Is toolName in DANGEROUS_TOOLS?
YES → Return true APPROVAL REQUIRED — no exceptions, no bypass for any user level.
NO → Continue to step 2.
2
Is toolName in OWNER_BYPASS_TOOLS?
YES → Branch on userLevel:
userLevel === 0 Return false ALLOWED — OWNER bypasses YELLOW tier.
userLevel > 0 Return true APPROVAL REQUIRED — team members need sign-off.
NO → Continue to step 3.
3
Is toolName in SAFE_TOOLS?
YES → Return false ALLOWED — execute immediately, no interruption.
NO → Continue to step 4.
4
Unknown tool — not in any tier.
DEFAULT → Return true APPROVAL REQUIRED — fail-safe. Unknown = approve. Never assume a new tool is safe. 
// TypeScript source — voltron-os-rules.ts
export function requiresApproval(toolName: string, userLevel: number): boolean {
  // Rule 1: Dangerous tools always require approval — no bypass, no exceptions.
  if (DANGEROUS_TOOLS.includes(toolName)) {
    return true
  }

  // Rule 2 + 3: Owner bypass tools — OWNER can proceed; everyone else needs approval.
  if (OWNER_BYPASS_TOOLS.includes(toolName)) {
    return userLevel !== 0
  }

  // Rule 4: Safe tools never require approval.
  if (SAFE_TOOLS.includes(toolName)) {
    return false
  }

  // Rule 5: Unknown tool — fail-safe. Require approval rather than assume safe.
  return true
}
