Warrior Launch Guide

5 steps. Every step is click-to-copy. If Talan can follow this, anyone can.

Step 0 — One-Time Setup (only do this ONCE per Mac)
If you've never connected to MDJ_SERVER from this Mac before, you need to add the SSH config. Open Terminal and paste this:

Add SSH config (one time only):

cat >> ~/.ssh/config << 'EOF'

Host mdj1-local
    HostName 192.168.6.93
    User jdm
    ForwardAgent yes
EOF
After this, ssh mdj1-local works forever from this Mac. Skip this step if you've already done it.
Step 1 — Open Terminal on your Mac

Open the Terminal app:

Spotlight (Cmd+Space) → type "Terminal" → hit Enter. Or find it in Applications → Utilities → Terminal.
Step 2 — SSH into MDJ_SERVER

Connect to the server (click to copy):

ssh mdj1-local
You'll see jdm@mdjserver:~$ when you're connected. If it asks for a password or says "connection refused," the server might be off.
Step 3 — Check for existing warrior sessions

See who's already running (click to copy):

tmux ls
This shows all active warrior sessions. If the warrior you want is already running, reattach with: tmux attach -t WARRIOR_NAME
Step 4 — Start a NEW warrior session
Only do this if the warrior ISN'T already running (Step 3 didn't show them). Pick ONE warrior below — click to copy, paste into terminal.

🥷 Launch SHINOB1 (CTO):
