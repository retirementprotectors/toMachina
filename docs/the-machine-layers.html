11 Layers — Why It Actually Works

The agents are Layer 8. The other 7 layers are what make Layer 8 possible. Remove any layer and the whole thing degrades.

1

Identity Most stop here

CLAUDE.md

Knows who JDM is, how RPI operates, how to communicate, what's forbidden, what's expected. Not a style guide — an operating manual for being YOUR engineer.

2

Standards

The Immune System

35 rules that BLOCK bad code before it's written. PHI never leaks. Credentials never get hardcoded. Insecure patterns are killed at the tool level. The system doesn't ask — it prevents.

3

Memory

Learning Loop

Carries decisions across sessions. Remembers corrections, sprint history, architecture choices. Promotes learnings overnight. Never repeats the same mistake twice.

4

Platform Knowledge Sophisticated stop here

toMachina

Knows every file, every route, every module, every package across 132K lines of typed platform code. Three portals, four shared packages, 68 API routes. The code IS the documentation.

5

Data Intelligence

ATLAS

Looks up any client, account, carrier, policy, or revenue record from 29K+ Firestore docs. Tracks every data source and how it flows. Streams analytics to BigQuery in real-time.

6

Tooling

MCP-Hub

82 tools across 9 servers. Sends emails, texts, and calls. Searches NPI registries, ICD-10 codes, Medicare coverage. Reads Google Drive, posts to Slack, books calendar events, designs in Canva. A brain with hands.

7

Process

FORGE

Takes a Discovery Doc through Plan → Build → Audit → Deploy → Verify. CI gates at every stage. E2E tests. Auto-merge. Branch protection. The process generates its own instructions.

8

Autonomy

RONIN

Takes a sprint ticket, builds the feature, passes CI, and ships to production — no human in the loop. Sprint 011: Discovery Doc to production in 2 hours 35 minutes. First shot. First try.

9

Self-Improving Infrastructure

The Closed Loop

Violations get logged, promoted to CLAUDE.md overnight, trend-tracked in compliance history. The system patches its own rules. Every session makes the next session smarter. The factory improves the factory.

10

Mesh Orchestration

The Dojo

6 warriors with named roles — CTO, COO, CMO, Guardian, Builder, Bot. Queue routing, war room comms, heartbeat monitoring, auto-triage. The agents manage the agent army.

11

Vision

Says what to build. Approves gates. Makes business decisions. The only human in the loop. The reason The Machine exists. The layer no AI can replace.
