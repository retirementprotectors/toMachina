What's varying

Same wordmark. Same tagline "We're Your People™". Same brand navy. What changes:

  • Vertical gap — tight / medium / loose spacing between wordmark baseline and tagline
  • Horizontal alignment — centered / right-aligned (under "Inc.") / left-aligned (under "Retirement")
  • Tagline size + weight — smaller SemiBold / medium SemiBold / larger Bold
  • Graphic device — with or without a divider line
  • Shield treatment — baked into existing wordmark (A–G) vs. scaled-up shield on the side (H)
