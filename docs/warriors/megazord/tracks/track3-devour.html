Failure Modes

Floating point drift in commissions: $100.10 stored as 100.09999999. Use integer cents internally or string comparison. Never compare currency with ===.

False dedup confidence: SUPER_MATCH too aggressive on name-only tier. "John Smith" in Des Moines matched to "John Smith" in Miami. Require additional signal (zip code, DOB) for name-only matches.

Stale cron: Staleness monitor itself goes stale (cron stops running, no one notices). Add meta-monitor: if staleness-monitor hasn't run in 48 hours, alert.
