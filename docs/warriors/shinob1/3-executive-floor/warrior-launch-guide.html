Warrior Launch Guide

Select your warrior. Follow the steps. Every command has a COPY button.

Step 0 — One-Time SSH Setup — only if you have NEVER connected before
Add this to ~/.ssh/config on your Mac. One time only.
Host mdj1-local HostName 192.168.6.93 User jdm ForwardAgent yes
After this, ssh mdj1-local works forever from this Mac.
Step 1 — Open Terminal

Open a new Terminal tab on your Mac

Cmd+Space → "Terminal" → Enter. One tab per warrior — up to 3 at once = 3 tabs.
Step 2 — SSH into MDJ_SERVER
ssh mdj1-local
You'll see jdm@mdjserver:~$ when connected.
