Next Steps — Document & Underwriting Requirements

Next Steps are fed directly by the recommendations. Each Health / Wealth / Legacy reco generates a specific set of document and underwriting requirements — this tab is the consolidated action plan.

How This Tab Gets Populated

STUB — this tab is derivative: populated automatically from the per-case recommendations. For each reco in Health / Wealth / Legacy:
  • Application forms — carrier new-business application, any supplemental questionnaires
  • Source-of-funds documentation — 1035 exchange forms, trustee-to-trustee IRA transfer forms, beneficiary designation forms, POD forms for bank CDs
  • Underwriting requirements — illustrations signed by client, suitability questionnaire, financial information worksheet, medical underwriting (for life/LTC if applicable)
  • Compliance documentation — replacement forms (if applicable), disclosure acknowledgments, carrier state-specific forms
  • Client-side items — church name + address for QCD designation, beneficiary contact info, POA document copies, estate document status
Each item is timestamped with who owns it (advisor / agent / client / carrier) and target completion date. Reference back to the specific Reco that generated it.
