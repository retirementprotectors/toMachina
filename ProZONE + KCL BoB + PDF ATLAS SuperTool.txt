    └── Assigned Territory
    └── Origin ZIP (drives tier math)
    └── TIER MAP (specialist-specific distance from origin → each zone/county)
    └── Schedule (Office Days / Field Days / slots per tier)
    └── Meeting Criteria (office vs field qualifiers)
    └── Zone Lead Criteria
    └── Calendar Booking Link
    └── Team (Coordinator, Associate1, Associate2)

  Two Admin Builders

  1. Territory Builder (define the geography)
  - Create/edit territories (name, state)
  - Assign counties to territories
  - Group counties into zones within each territory
  - This is the "map" — it doesn't know about specialists

  2. Specialist Config (assign people to territories)
  - Select specialist → assign to territory
  - Set origin ZIP → system calculates tiers per zone/county
  - Configure office days (Mon/Fri), field days (Tue/Wed/Thu)
  - Set slot templates per tier (the time grids from the sheet)

  - Link calendar booking URL
  - Assign coordinator + associates
