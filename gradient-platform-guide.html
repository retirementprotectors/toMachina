Path A — Advisory (Gradient Wealth Management)

PATH A: Advisory — Gradient Wealth Management (GWM)

Josh acts as GWM Investment Advisor Representative (IAR). All Path A cases use GWM TAF + GWM IAA. Three sub-paths differ by TPMM and custodian:

A1 — Gradient Investments (GI) ⭐ Most Common

  • GWM as RIA + GI as TPMM
  • Custodian: Charles Schwab
  • Access to all 41 GI strategies (ETF Endowment, Tactical, Buffer, etc.)
  • Forms: GWM TAF + IAA + GI IPC (Schedules A–D) + Schwab forms
  • TPM Fee: 0.40% (standard) to 1.00% (Gradient 50)

A2 — AssetMark TAMP

  • GWM as RIA + AssetMark as TPMM
  • AssetMark platform and custody
  • AssetMark product lineup (not GI strategies)
  • Forms: GWM TAF + IAA + AssetMark application
  • Used when GI products don't fit client profile

A3 — Orion TAMP

  • GWM as RIA + Orion as TPMM
  • Custodian: Charles Schwab
  • Orion platform strategies
  • Forms: GWM TAF + IAA + Orion application + Schwab forms
  • Alternative TAMP option via GWM
Path B — Securities (Gradient Securities BD)

PATH B: Securities — Gradient Securities (GS BD)

Josh acts as GS Series 7 General Securities Representative. Commission-based. No IAA. No ongoing advisory fee. Two sub-paths:

B1 — BD Brokerage (RBC)

  • Brokerage accounts held at RBC Capital Markets
  • Used when Schwab brokerage access isn't available
  • Commission-based — no ongoing advisory fee
  • Forms: RBC TAF + RBC CAF + RBC Opening Kit + RBC Transfer Form
  • GI portfolio strategies are NOT available through RBC

B2 — Direct Products

  • Variable Annuities, Mutual Funds, REITs, 401ks
  • Products go direct to carrier — no RBC/Schwab custodian
  • Commission-based
  • Forms: Carrier-specific TAF + carrier application + carrier forms
  • Each carrier has its own form set
🚨 Rules That Cannot Break
  • GI strategies (ETF Endowment, BIP, etc.) ONLY through Schwab (Path A1)
  • When brokerage is needed and Schwab isn't available → RBC for brokerage only
  • Advisory (GWM/IAR) and Brokerage (GS Series 7) are ALWAYS separate — separate TAFs, separate forms, separate submissions
  • Never list "Gradient Investments" as the vendor on an RBC form

When a Client Has Multiple Accounts
  • Each account gets its own strategy based on that account's individual balance vs minimums

NIGO Prevention — Real Errors from Real Cases
These All Actually Happened at RPI

Clark #570593 and Stirratt #571086 flagged by Morgan Smick. Mark Grams put a hold on all applications. Not hypothetical.

1. GI Paperwork on an RBC Case

Stirratt — original NIGO, Case #571086

Submitted GI/Schwab advisory forms for an RBC brokerage account.

RBC cases use ONLY RBC forms. GI IPC, IAA, GI Disclosure — never for RBC.

2. Wrong Dollar Amount — Checking Balance ≠ IRA Balance

Stirratt — current packet

TAF shows $226,424.38 (checking balance from CAF). IRA being transferred is $223,955 at Schwab #31846959.

Source of Funds amount = the specific account balance being transferred. Get it from the account statement.

3. Wrong Vendor on TAF ("Brokerage RBC")

Stirratt — current packet

RBC is the receiving firm, not the delivering firm. Money is coming FROM Schwab.

Vendor = DELIVERING firm. For Schwab transfers: "Charles Schwab." For Pershing: "Pershing LLC."

4. Blank Beneficiary on IRA Adoption Agreement

Stirratt — current packet

IRA without a named beneficiary defaults to the estate.

Always collect: primary beneficiary name, relationship, SSN, DOB, percentage before building the form.

5. Primary Residence Value Missing from CAF

Stirratt — current packet

Net worth shows $1.4M but residence is blank. Math doesn't work. GS will flag it.

Net worth = investable assets + residence value − liabilities. All three must be filled and must reconcile.

6. Date Mismatch Across Forms

Stirratt — current packet

Rep Ack page: CAF dated 03/17. Client Ack page: 02/17. Client acknowledges the wrong document.

After building any packet, read every date box. All CAF date references must match. All COMRA date references must match.

7. Whiteout or Document Alteration

Clark #570593 — LPOA revoked, compliance hold

Whiteout detected on a signed document. Automatic compliance risk in a FINRA environment.

Zero tolerance. Error before signing → reprint. Error after signing → new form, new signature.

8. Funds Transferred Before Approval

Clark + Budd — flagged by Mark Grams

$14,470 transferred for Clark without approval. $100K uninvested 27+ days for Budd.

GS approves → client signs → then funds move. This order is fixed. No exceptions.

9. Contradictory Investment Objectives

Both "Preservation of Capital" and "Growth and Income" checked. Pick one.

Check ONLY the objective that matches the COMRA category and Best Interest narrative.

10. Fee Stated as 1.53% Instead of 1.40%
