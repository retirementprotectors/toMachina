Full Integration Scope

Everything Josh can do with Claude Code today, the MDJ can do through the web interface. Same APIs, same OAuth, same capabilities — minus dev tools the team doesn't need.

Capability Integration Example MDJ Command

Calendar Google Calendar API "Schedule a call with the Parmenter team for Thursday at 2pm"

Compliance CAM + OS "What compliance items are overdue for my team?"

No Local Installation Required

Today, these integrations run as local MCP servers on Josh's machines. The MDJ runs them server-side from Cloud Run. The data was never on the laptop — only the OAuth tokens and middleware code were. Those move to Cloud Run (environment variables + Secret Manager). The team opens a browser and types. That's it.
