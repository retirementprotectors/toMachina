Full Integration Scope

Everything Josh can do with Claude Code today, the MDJ can do through the web interface. Same APIs, same OAuth, same capabilities — minus dev tools the team doesn't need.

Capability Integration Example MDJ Command
EmailGmail API"Email Henderson about their RMD, CC Nikki, show me the draft first"
CalendarGoogle Calendar API"Schedule a call with the Parmenter team for Thursday at 2pm"
DocumentsGoogle Drive API"Pull the latest commission statement from the Gradient shared drive"
MessagingSlack API"Send Vinnie a Slack message that the Gradient book is imported"
Calls + SMSTwilio API"Text Henderson a reminder about their annual review next Tuesday"
Healthcare DataNPI + ICD-10 + CMS"Check if Dr. Martinez has an active NPI and what specialties"
Pipeline MgmtFlow Engine API"Move the Henderson case to underwriting and notify the team"
Client DataFirestore + API"Find all clients with Humana MAPD in zip 50309"
CommissionsRevenue API"Show me this month's commission breakdown by carrier"
ComplianceCAM + OS"What compliance items are overdue for my team?"

No Local Installation Required

Today, these integrations run as local MCP servers on Josh's machines. The MDJ runs them server-side from Cloud Run. The data was never on the laptop — only the OAuth tokens and middleware code were. Those move to Cloud Run (environment variables + Secret Manager). The team opens a browser and types. That's it.
