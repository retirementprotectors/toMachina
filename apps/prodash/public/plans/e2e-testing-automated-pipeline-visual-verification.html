Overview

toMachina has shipped 674 items across 9 sprints with zero automated test infrastructure. CI currently covers only type-check (13/13) and build (11/11). Runtime behavior regressions, broken intake pipelines, and UI rendering failures are invisible to CI. 470 items require manual CEO visual verification after every deploy.

TicketHelperExportsKey Detail

#RiskMitigation
1Claude Vision costs — 4 tests x ~2 Vision calls = ~8 API calls per CI runCost is minimal (~$0.10/run). Monitor monthly spend.
2Flaky Drive operations — Google Drive API has occasional latency spikesPoll helper's 60s timeout accommodates this. Retry logic in drive-client.
3Auth token expiry — Custom tokens last 1 hourCI runs complete well within window. Cache invalidation on 401.
4Shared Drive permissions — CI service account needs Editor accessGrant during Phase 4 setup. Document in POSTURE.md.
