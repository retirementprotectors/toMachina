❯ So the wire data import has super tools in five stages, and each of those

⏺ Yes. Three layers deep:

  1. WIRE_DATA_IMPORT — the one pipeline you kick off
  2. 5 SUPER_TOOLS — the stages (Extract, Validate, Normalize, Match, Write)
  3. ATOMIC_TOOLS — the individual workers inside each super tool

  - Phone validator (type, carrier, fake detection)
  - NeverBounce (email deliverability)
  - USPS Address v3 (CASS-certified standardization)
  - Contact quality scorer (composite A-F grade)

  You never see layer 3. The super tools pick which atomic tools to run based on
   what the data needs.

  One button. Three layers. Everything fires.
