Step 0 — One-Time SSH Setup — only if you have NEVER connected before
Add this to ~/.ssh/config on your Mac. One time only.
Host mdj1-local HostName 192.168.6.93 User jdm ForwardAgent yes
After this, ssh mdj1-local works forever from this Mac.

tmux Commands
